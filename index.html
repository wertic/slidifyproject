Presentation agenda

The presentation has been created as a part of Developing Data Products class project to exsercise gained skills and knowledge in creating the products ready to be distributed and reuse for general audience. The fisrt part of the class project is creating the app using Shiny and the second is applying Slidify to create a presentation which is going to be give a persation of Shiny app.

The description of the app

  1. The app is created to do prediction of mpg based on Mtcars dataset.
  2. UI elements include: tabs, panels, checkboxes, sliders, histograms.
  3. Linear model is hidden and it calculates the outcome very quickly.

The app is available here: https://wertic.shinyapps.io/Mtcars_dataproduct
